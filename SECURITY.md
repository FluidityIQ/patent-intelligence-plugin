# Security Policy

## Supported Versions

We provide security updates for the current major version of the Patent Intelligence Plugin. Please upgrade to the latest release to receive security fixes.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in the Patent Intelligence Plugin or in the FluidityIQ Patents MCP service it connects to, please report it responsibly.

**Do not** open a public GitHub issue for security vulnerabilities.

- **Email:** security@fluidityiq.com
- **Scope:** This policy covers the patent-intelligence-plugin repository and the FluidityIQ Patents MCP server (patents.fluidityiq.ai) used by the plugin.

We will acknowledge your report and aim to respond within 5 business days. We will keep you informed of our progress and may ask for additional information. We will disclose the issue publicly after a fix is available, unless we agree otherwise.

Thank you for helping keep our users safe.
